\section{ECE 537: Foundations of
Computing}\label{ece-537-foundations-of-computing}}

\hypertarget{homework-1-hire-assistant-problem}{%
\subsection{Homework \#1: Hire-Assistant
Problem}\label{homework-1-hire-assistant-problem}}

Write a program in the language of your choice (C++, Python, etc.) that
generates all 8! permutations of the array A = \textless{} 1, 2, 3, 4,
5, 6, 7, 8 \textgreater{} and computes the number of ``hires'', H1, H2,
\ldots{} as described in the lecture. Then compute the theoretical
expected value as EH = 1/8!(H1 + H2 + \ldots{} + H8!) and the
theoretical variance as Var(H) = E(H\^{}2) - (EH)\^{}2 = 1/8!(H1\^{}2 +
H2\^{}2 + \ldots{} + H81\^{}2) - (EH)\^{}2. Compare the output of your
program to the value for EH found in the lecture. Compare also to the
value ln (8).
